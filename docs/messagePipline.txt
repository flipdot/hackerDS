Message pipleline pathes
========================

core            --> sends a message to the core server
core.controller --> sends a message to the core controller page
core.display    --> sends a message to the core display page

apps.HelloWorldApp            --> sends a message to the HelloWorldApps server
apps.HelloWorldApp.controller --> sends a message to the controller of the HelloWorldApp
apps.HelloWorldApp.display    --> sends a message to the display of the HelloWorldApp


Javascript methods to access the pipeline
=========================================

core.send()            --> sends a message to the core server
core.controller.send() --> sends a message to the core controller page
core.display.send()    --> sends a message to the core display page

app.send()            --> sends a message to the current apps server
app.controller.send() --> sends a message to the current apps controller
app.display.send()    --> sends a message to the current apps display
